# Superstore Sales 2015 - 2018

## Project Overview
This project analyzes a retail sales dataset to evaluate business performance using key performance indicators (KPIs) and exploratory data analysis (EDA). An interactive Tableau dashboard was built to visualize sales trends, category performance, and geographic distribution.

## Dataset
- Source: Kaggle (Superstore Sales Dataset)
- Records: ~9,8000
- Key Columns: Sales, Profit, Category, Region, Order Date, Country, Segment etc.

## KPIs
- Total Sales
- Total Sales by Category
- Total Sales by Region
- Overall Sales by Month and Year
- Hiegest Sales by City
- Top Product by Sales
- customer segment contributes most
- Seasonal Pattern Month to Month and Year
- Seasonal Pattern for 2015
- seasonal pattern of 2016
- seasonal pattern of 2017
- seasonal pattern of 2018
- Ship mode is most used
- Sales underperforming geographically

## Tools Used
- Python (Pandas, Matplotlib) for EDA
- Jupyter Notebook
- PowerBI for Dashboard Visualization

## Key Insights
- The West region contributes the highest sales among all regions.
- Technology category generates the highest revenue.
- Sales show seasonal peaks toward the end of the year.


## Conclusion
This project demonstrates end-to-end data analysis from raw data exploration to business-focused visualization and insights.
